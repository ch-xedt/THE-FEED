THE FEED is a personal news dashboard that pulls live headlines from independent sources and presents them in a single, distraction-free interface.<br>
There is no algorithm deciding what you see first. There is no engagement optimization. There is no tracking. Just the news, straight from the source.

&nbsp;


<pre>No framework       Pure HTML, CSS and vanilla JavaScript </pre>
<pre>No backend         Runs entirely in the browser </pre>
<pre>No tracking        Zero cookies, zero analytics, zero data collection </pre>
<pre>No algorithm       Feeds sorted strictly by publication time </pre>
<pre>Responsive         Works seamlessly on desktop, tablet and mobile </pre>
<pre>Live data          Every visit fetches fresh headlines </pre>
<pre>Theme support      Toggle between stunning light and dark mode </pre>


&nbsp;

 ---
 
Information should be accessible without friction. And it should feel that way.<br>
THE FEED does one thing: it shows you what was published, when it was published, and links you directly to the source.<br>
A free press depends on people actually reading it. Aggregating public RSS feeds is not a replacement for journalism...it is a small act in favor of it. <br>
